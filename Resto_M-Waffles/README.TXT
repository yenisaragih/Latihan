USER INSTRUCTIONS           : Jalankan program Menu. Jika ingin memesan makanan klik ORDER kemudian masukkan
                              jumlah setiap makanan dan minuman yang akan di pesan. Klik COUNT untuk menghitung
                              total harga makanan dan minuman yang ada pesan. Masukkan uang anda di kolom YOUR MONEY
                              lalu klik YOUR CHANGE. Jika muncul "It's not enough!" berarti uang anda kurang dari 
                              harga pesanan anda. Kemudian klik RESET jika ingin memulai kembali. Klik EXIT jika ingin
                              melihat kembali daftar menu.
                              Klik CLOSE untuk menutup program.
                              
NOTE TAMBAHAN               : Program ini menggunakan font yang saya download sendiri. Apabila font ini tidak terdapat
                              pada PC user harap install font yang ada di folder untuk menghindari tampilan yang tidak lengkap.
